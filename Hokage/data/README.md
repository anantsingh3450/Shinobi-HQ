# Runtime Data

Gitignored. Bot outputs and execution artifacts stored here at runtime.

| Folder | Contents |
|--------|----------|
| `research/` | Research Bot reports |
| `backtests/` | Backtest results and charts |
| `risk/` | Risk assessment reports and pass/fail verdicts |
| `execution/paper/` | Paper trade logs and virtual portfolio snapshots |
| `execution/live/` | Live trade logs and fill records |

This folder is created automatically when bots run.
