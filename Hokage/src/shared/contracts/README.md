# Contracts

Canonical data schemas exchanged between bots.

Planned schemas:

- `StrategySpec` — output of Strategy Bot
- `ResearchReport` — output of Research Bot
- `BacktestResult` — output of Backtest Bot
- `RiskAssessment` — output of Risk Bot (paper and live verdicts)
- `LiveExecutionGate` — Hokage record of user confirmation, risk pass, broker ready
- `OrderIntent` — output of strategy engine
- `PipelineState` — workflow stage tracked by Hokage

No schemas defined yet.
