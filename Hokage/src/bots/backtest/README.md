# Backtest Bot

Validates strategy specs against historical data before risk assessment and execution.

## Inputs

- Strategy spec from Strategy Bot
- Historical data via `integrations/data/`

## Outputs

- Backtest report (metrics, drawdown, trade log) → `data/backtests/`
- Handoff to Risk Bot for pre-execution assessment

## Implementation
The Backtest Bot is implemented in `backtest_bot.py`. It uses `HistoricalBacktestEngine` (in `engine/historical_backtest_engine.py`) to simulate strategy execution over historical candles generated by a data provider. The bot records strategy performance to the predictions ledger (`predictions.jsonl`) via `JsonPredictionLedger`.
